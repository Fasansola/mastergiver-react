# MasterGiver v1 - Milestone Roadmap

## ✅ Milestone 0: Foundation & Infrastructure (COMPLETED)

- Next.js 14+ setup with TypeScript
- Prisma + PostgreSQL configured
- Database seeded with causes/skills
- Folder structure established
- All dependencies installed

## 🎯 Milestone 1: Authentication & User Management (CURRENT - Week 3-4)

- Sign-up page (email/password)
- Login page
- Email verification flow
- Password reset flow
- Session management via Auth.js
- Protected route middleware
- Username uniqueness validation

## 📋 Milestone 2: Profile Foundation & Dashboard (Week 5-6)

- Dashboard layout
- Profile picture upload (2MB, GIF/JPG/PNG)
- Profile form (name, username, city, state, aboutMe)
- Profile preview component
- Save as draft functionality
- Username locked after first save

## 🎨 Milestone 3: Onboarding Flow (Week 7-8)

- 6-step guided onboarding
- Progress auto-save
- Skip and resume functionality
- "Launch Profile" action

## 🎯 Milestone 4: What I Care About & Giving Skills (Week 9-10)

- Select predefined causes (with icons/colors)
- Add custom causes
- Select/add giving skills
- Display on profile

## 🏢 Milestone 5: Organizations Integration (Week 11-12)

- Pledge.com API integration
- Organization search
- Manual organization addition
- Default logo for manual orgs

## 🌐 Milestone 6: Public Profile Pages (Week 13-14)

- Public route: /[username]
- SEO optimization (Open Graph, Twitter Cards)
- Draft vs Published logic
- Social sharing

## 🔧 Milestone 7: Admin Panel (Week 15-16)

- Admin dashboard
- Manage causes (add/remove)
- Manage skills (add/remove)
- Icon/color picker UI

## 🚀 Milestone 8: Polish & Launch (Week 17-18)

- Accessibility audit
- Performance optimization
- Error handling
- Deploy to Vercel
